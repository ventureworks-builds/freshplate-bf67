# MealWeek

Your family's week, planned in minutes.

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.local.example` to `.env.local` and fill in your values
3. Set up your Supabase project and run the SQL in `setup.sql`
4. Run `npm run dev` to start locally
5. Deploy to Vercel: `vercel deploy`

## Template

Built with the VentureWorks **membership** template.

## Customisation

Edit `config.ts` to update:
- Product name, tagline, colors
- All text content
- Email addresses, links

No code changes needed for content updates.
